## 1. 游戏定位

### 1.1 游戏类型

**地块召唤师 (TileSummoner)** 是一款融合多种核心玩法的对战游戏：

| 核心玩法 | 占比 | 说明 |
|----------|------|------|
| 地形↔单位转换 | 50% | **核心**：消耗地块召唤单位，地块是单位的物质基础 |
| 即时对战 | 30% | 类似《皇室战争》的实时对战 |
| Roguelike爬塔 | 15% | 类似《杀戮尖塔》的局外成长 |
| 技能卡牌 | 5% | 卡牌作为法术/增益/减益等辅助手段 |

### 1.2 核心体验一句话

> **"地块与单位的转换是核心——在任何己方控制区域消耗地块即可召唤单位；卡牌是大局应急消耗品，用于突破上限、保护下限、紧急解围，使用后永久消失。"**

### 1.3 玩法结构

```
大局（Roguelike爬塔）- 3个Act共45层，每层含3-4个可选节点
│
├─ Act 1（第1-15层）- 基础单位为主
├─ Act 2（第16-30层）- 新增中型单位
└─ Act 3（第31-45层）- 全套单位池
│
├─ 节点类型：[战斗] [商店] [休息] [升级] [精英] [?] [Boss]
└─ 失败 = 游戏结束
```

### 1.4 目标平台

- **PC端**: Steam

## 2. 核心特色

### 2.1 独特卖点 (USP)

| 特色 | 描述 | 差异化 |
|------|------|--------|
| **地形作为资源** | 地块是可消耗、可编辑的战斗资源 | 区别于传统卡牌游戏的固定战场 |
| **连通性博弈** | 必须维持与敌方的连通，断路即判负 | 独创机制，增加策略深度 |
| **虚空坠毁** | 切断敌方路径可使敌人坠落消失 | 攻防一体，高风险高收益 |
| **控制带推进** | 战线前移可就地征用敌方地块 | 动态资源争夺，鼓励进攻 |

### 2.2 核心循环

**小局内循环**（单场对战）：
```
选择地块 → 消耗地块召唤单位 → 单位推进战线 → 争夺控制区域
                ↑                                  ↓
                └── 使用卡牌（应急） ──────────────────┘
                      (突破上限/保护下限/紧急解围)
```

**大局循环**（爬塔进程）：
```
节点选择 → 小局对战 → 胜利/失败 → 获得奖励 → 选择奖励 → 进入下一层
                                       ↓
                                    (金币/卡牌/地块/遗物)
                                        ↓
                                    失败 → 游戏结束
```

**卡牌使用循环**（应急消耗品）：
```
大局卡池（40种）→ 选择携带（15-20张）→ 战斗栏位（6-8张）
                                              ↓
                                       使用 → 永久消失
                                              ↓
                                     战斗结束不恢复，需重新获取
```

### 2.3 核心机制说明

| 机制 | 说明 | 定位 |
|------|------|------|
| **地块↔单位转换** | 消耗地块召唤单位，地块消失 | **核心** |
| **连通性博弈** | 维持己方基地与敌方连通，断路判负 | 核心约束 |
| **控制带争夺** | 推进战线，在敌方区域召唤单位 | 扩张机制 |
| **卡牌应急** | 法术、增益、减益等效果，大局消耗品 | **应急辅助** |

> **卡牌定位说明**：
> - 卡牌是**大局应急消耗品**，用于突破上限、保护下限、紧急解围
> - 使用后永久消失，战斗结束不恢复
> - 5种使用类型：×3（基础生存）、×2（战术调整）、单场1次、本场持续、自动触发、整局1次
> - 详见 [04_卡牌系统](./04_卡牌系统.md)

### 2.4 角色系统

新游戏可选择角色，每个角色有独特的初始地形配置和被动技能：

- **元素师艾琳** - 元素连锁风格，相邻不同地形单位获得攻击加成
- **守卫官雷恩** - 阵地防御风格，控制区内单位获得防御加成
- **游侠希尔** - 机动突进风格，森林地形单位获得移速加成

> 详细角色设计见 [09_元游戏系统](./09_元游戏系统.md)

### 2.5 局外成长系统

采用**成就解锁制**：每局游戏从零开始，达成成就可永久解锁新卡牌/地块/角色。

> 详细成就列表见 [09_元游戏系统](./09_元游戏系统.md)

### 2.6 每日挑战模式

固定种子 + 固定初始配置 + 每日特殊规则，所有玩家同一起跑线竞争。

> 详细规则见 [09_元游戏系统](./09_元游戏系统.md)

## 3. 目标用户

### 3.1 核心用户画像

| 用户类型 | 特征 | 需求 |
|----------|------|------|
| **皇室战争玩家** | 喜欢快节奏即时对战 | 3-5分钟一局，碎片化时间 |
| **杀戮尖塔玩家** | 喜欢爬塔、构筑流派 | 局外成长、遗物组合 |
| **策略玩家** | 喜欢深度思考、计算收益 | 复杂的连通性博弈 |
| **创意型玩家** | 喜欢寻找非常规解法 | 地形编辑带来的战术多样性 |

### 3.2 年龄段与地域

- **主要年龄段**: 18-35岁
- **重点地区**: 中国、东南亚、欧美

## 5. 设计哲学

### 5.1 易学难精

- **3分钟上手**: 基础操作简单直观
- **百小时精通**: 连通性博弈与地形组合深度

### 5.2 反制与平衡

- 每种战术都有明确反制手段
- 避免无脑最优解，鼓励思考

### 5.3 正反馈循环

- 地形变化即时可见
- 控制带切换有爽快反馈
- 断路预警增加紧张刺激

---

## 6. 商业化方向

### 6.1 付费模式

- **买断制**: 基础游戏一次性购买
- **DLC扩展**: 新阵营、新地形、新单位
- ** cosmetic道具**: 皮肤、表情、棋盘主题

### 6.2 避免设计

- 不影响平衡性的付费优势
- 不强制消费的卡牌获取
