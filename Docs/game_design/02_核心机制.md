## 1. 机制概述

《地块召唤师》的核心机制是**地块与单位之间的转换**，辅以**卡牌应急系统**。

> **核心定位**：
> - **地块 ↔ 单位转换**：消耗地块即可召唤单位，地块是单位的物质基础，这是游戏的核心玩法
> - **卡牌是应急消耗品**：卡牌作为法术/增益/减益等消耗品，用于突破上限、保护下限、紧急解围，使用后永久消失
> - **连通性是纯地形属性**：连通性判定只看地块，与单位完全无关

玩家可以在任何己方控制区域选择地块并消耗它来召唤单位。召唤后地块消失，单位诞生在相邻地块上。同时需要维持己方阵地与敌方基地的连通——断路将导致己方单位坠入虚空判负。

### 1.1 双层资源机制

```
┌─────────────────────────────────────────────────────────┐
│  大局资源（跨小局累积，永久保留）                         │
│  ├─ 卡牌池：携带15-20张，战斗中选择6-8张                │
│  ├─ 地块库存：商店购买、事件奖励                         │
│  ├─ 遗物：被动/主动技能                                 │
│  └─ 金币：用于购买卡牌/地块                              │
├─────────────────────────────────────────────────────────┤
│  小局资源（战斗开始时重置）                               │
│  ├─ 战场地块：从库存配置，战斗中消耗后本场消失           │
│  └─ 法力值：初始5点，每秒+1，上限10点                   │
└─────────────────────────────────────────────────────────┘
```

### 3.2 控制区规则

| 规则 | 说明 |
|------|------|
| **范围** | 第5、6、7、8行（从底部向上数4行），共 4×7 = 28 格 |
| **编辑权限** | 玩家可在此区域内自由编辑地块 |
| **初始状态** | 游戏开始时，此区域由我方完全控制 |
| **敌方对应区** | 敌方同样拥有第0、1、2、3行的4×7控制区 |

### 3.3 地块编辑方式

玩家可通过以下方式操作地块：

| 方式 | 来源 | 效果 |
|------|------|------|
| **商店购买** | 局外金币消费 | 购买特定地形地块，替换控制区内地块 |
| **事件奖励** | 层间随机事件 | 获得稀有地形地块或地块改造道具 |
| **遗物效果** | 装备的遗物 | 被动提供地块加成或主动改造地块 |
| **卡牌技能** | 打出法术卡 | 临时改变地块属性（战斗结束后恢复） |

### 3.4 单位元素词条（8种）

每个单位拥有一个元素词条属性，与所在地块类型发生交互：

| 词条 | 对应地块 | 描述 |
|------|----------|------|
| **地** | 农田、沙地 | 土系相关单位 |
| **风** | （暂无对应地块） | 机动性相关单位 |
| **水** | 水域、沼泽 | 水系相关单位 |
| **火** | 熔岩 | 火系相关单位 |
| **草** | 草地、森林 | 草系相关单位 |
| **金** | 岩石 | 金系相关单位 |
| **冰** | 冰原 | 冰系相关单位 |
| **通用** | （无对应地块） | 无明显加成减益 |

### 3.5 地块类型与特性分类

地形按**特性复杂度**分为三类，每种地形都有独特效果和对应单位词条：

| 分类 | 地形 | 对应词条 | 特点 | 获取方式 |
|------|------|----------|------|----------|
| **基础地形** | 草地 | 草 | 无特殊效果，基准地形 | 初始携带 / 商店低价 |
| | 水域 | 水 | 非水词条移动力-1，溺水5%/秒 | 初始携带 / 商店低价 |
| | 沙地 | 地 | 非地词条移动力-1，地词条移动力+1，视野+1 | 初始携带 / 商店低价 |
| **进阶地形** | 岩石 | 金 | 防御+50%，消耗后地块保留 | 商店中价 / 事件奖励 |
| | 森林 | 草 | 草词条防御+20%，20%闪避，飞行无视 | 商店中价 / 事件奖励 |
| | 农田 | 地 | 每5秒恢复5%生命，防御-10%，草词条移动力+1 | 商店中价 / 事件奖励 |
| **特殊地形** | 熔岩 | 火 | 非火词条灼烧10%/秒，火词条攻击+20% | 商店高价 / 精英层奖励 / 遗物 |
| | 沼泽 | 水 | 非水词条移动力-1，水词条移动力+1，困住攻击者15% | 商店高价 / 精英层奖励 / 遗物 |
| | 冰原 | 冰 | 非冰词条移动力-1，冰词条移动力+1，冻结10% | 商店高价 / 精英层奖励 / 遗物 |

> **设计原则**: 基础地形简单直观，进阶地形有明确优缺点，特殊地形有强大效果但伴随风险。

## 4. 连通性机制（核心）

### 4.1 主路规则

> **连通性是纯地形属性，与单位完全无关。**

| 规则 | 说明 |
|------|------|
| **判定依据** | 仅地块的4向相邻（上、下、左、右） |
| **斜向** | 不构成连通 |
| **单位参与** | ❌ 地面单位和飞行单位都不参与连通性判定 |
| **要求** | 必须存在一条从己方主基地连通至敌方主基地的地块路径 |

> **必须存在一条从己方主基地连通至敌方主基地的路径。**

### 4.2 断路判负

| 条件 | 结果 |
|------|------|
| 操作后不存在从己方基地到敌方基地的连通路径 | **立即判负** |

**设计意图**: 防止玩家无限制掘地/改路，平衡攻防策略

### 4.3 虚空坠毁

当发生以下情况时，相关地形格及其上的非飞行单位将"坠入虚空"：

1. **地形消失**: 某个地块被消耗/移除
2. **连通性检查**: 以"主路"为基准进行BFS/DFS搜索
3. **坠落判定**: 无法连通回主路的地形格及其上的非飞行单位坠落消失

**战术意义**: 鼓励高风险高收益的"掘地"策略

### 5.2 夺取规则

| 条件 | 结果 |
|------|------|

| 我方控制区（第5-8行） | 自由召唤，无限制 |
| 中线区域（第4行） | 需争夺控制权 |
| 敌方半区（第0-3行） | 夺取后可就地召唤 |

## 7. 经济系统

### 7.1 资源类型

| 资源 | 获取方式 | 主要用途 |
|------|----------|----------|
| **地块** | 初始持有 / 商店购买 / 争夺控制带 | **核心资源**：消耗地块召唤单位（地块↔单位转换） |
| **法力值** | 每秒自动回复，初始5点，上限10点 | 打出卡牌（应急）的消耗 |
| **卡牌** | 局外卡池（40种）/ 对战奖励 | **应急消耗品**：法术、增益、减益等效果（大局消耗品） |
| **遗物** | 层间奖励 / Boss掉落 | 被动技能或主动技能，定义战术流派 |
| **金币** | 对战产出 | 局外购买卡牌、地块、升级 |

> **核心定位说明**：
> - **地块是核心资源**：消耗地块即可召唤单位，这是游戏的核心玩法
> - **卡牌是应急消耗品**：
>   - 用于突破上限、保护下限、紧急解围
>   - 使用后永久消失，战斗结束不恢复
>   - 5种使用类型：×3、×2、单场1次、本场持续、自动触发、整局1次
> - 召唤单位不需要卡牌，只需要选择地块并消耗它

### 7.2 卡牌使用机制

卡牌是**大局应急消耗品**，使用规则如下：

| 使用类型 | 说明 | 示例 |
|----------|------|------|
| **×3** | 可在单场战斗中使用3次 | 急救包、法力换血 |
| **×2** | 可在单场战斗中使用2次 | 紧急传送、群体加速、神圣庇护 |
| **单场1次** | 每场战斗可使用1次 | 清场术、狂暴、破法之刃 |
| **本场持续** | 非消耗品，整场战斗持续生效 | 法力暴涨、地块扩增、地形精通 |
| **自动触发** | 满足条件自动触发，不计次数 | 基地守护、断路保险 |
| **整局1次** | 整个Roguelike大局只能使用1次 | 防斩杀、复活守护、时间回溯 |

> **平衡性设计原则**：
> - 整局一次卡牌提供**优势**而非**翻盘**（如防斩杀以1点存活而非满血复活）
> - 基础应急卡（低费）提供多次使用，给予容错空间
> - 强力大招（中高费）单场一次，作为关键转折点

### 7.3 召唤单位机制

**召唤流程**（三步）：

```
步骤1: 选择地块
    ↓ 点击任何己方控制的地块
步骤2: 选择单位
    ↓ 弹出该地块对应的单位选项（每种地形3种）
步骤3: 选择召唤位置
    ↓ 选择该地块相邻的一个正常地块（不能是虚空）
结果: 被点击地块消失（变为虚空），单位诞生在相邻地块上
```

**召唤规则**：
- 可以选择任意己方控制的地块（包括敌方半区已夺取的控制带）
- 相邻地块必须是正常地块（虚空不是地块，不能放置）
- 召唤后地块消失，需注意连通性规则
- 每种地形有3种可选单位，定位由地块属性决定
- 召唤有短暂的施法时间

### 7.4 法力系统

| 属性 | 数值 | 说明 |
|------|------|------|
| 初始法力 | 5 点 | 战斗开始时的法力值 |
| 回复速度 | 每秒 +1 点 | 正常情况下的自动回复 |
| 法力上限 | 10 点 | 常规战斗中最大值 |
| 加时回复 | 每秒 +2 点 | 3分钟后进入加时赛，双倍回复 |
| 突破上限 | 卡牌可临时或永久突破 | 部分卡牌允许超上限使用法力 |

## 8. 地形与单位系统

### 8.1 地块↔单位转换（核心）

这是游戏的核心机制。消耗地块即可召唤单位，地块是单位的物质基础。

```
地块 ───[消耗]──→ 单位
  ↓                 ↓
消失            战场存在
```

**召唤流程**（三步）：
1. **选择地块**：点击任何己方控制的地块
2. **选择单位**：弹出该地块对应的单位选项（每种地形3种）
3. **选择位置**：选择该地块相邻的一个正常地块（不能是虚空）

**结果**：被点击地块消失（变为虚空），单位诞生在相邻地块上。

### 8.2 地形类型与对应单位（9种）

每种地形可生成3种单位，单位定位由地块属性决定：

| 地形 | 对应词条 | 可选单位定位 |
|------|----------|-------------|
| 水域 | 水 | 移动限制，减益效果 |
| 沙地 | 地 | 移动减慢，视野优势 |
| 岩石 | 金 | 防御加成，固定障碍 |
| 草地 | 草 | 均衡通用 |
| 森林 | 草 | 闪避加成，隐蔽效果 |
| 农田 | 地 | 持续恢复 |
| 熔岩 | 火 | 伤害加成，持续灼烧 |
| 沼泽 | 水 | 控制效果，减速 |
| 冰原 | 冰 | 冻结效果，减速敌人 |

### 8.3 地形编辑

玩家可在控制区内对地块进行：

- **召唤单位**: 消耗地块召唤单位（地块消失，需注意连通性）
- **局外编辑**: 通过商店、事件、遗物等方式更换控制区内地块类型
- **临时改造**: 使用法术卡临时改变地块属性（战斗结束后恢复）

## 9. 单位系统

### 9.1 单位属性

| 属性 | 说明 |
|------|------|
| 生命值 (HP) | 单位存续能力 |
| 攻击力 (ATK) | 对敌方造成的伤害 |
| 移动力 (MOVE) | 每回合可移动格数 |
| 攻击范围 (RANGE) | 攻击距离 |
| 地形亲和 | 在特定地形上的加成 |

### 9.2 单位词条与地块交互

每个单位拥有一个元素词条属性，当单位站在不同地块上时，会触发不同档位的属性加成：

| 亲和档位 | 加成效果 | 触发条件 |
|----------|----------|----------|
| **+50%** | 极大优势 | 单位词条与地块完全匹配的特殊情况 |
| **+20%** | 主场优势 | 单位词条对应的地块（如水词条站在水域） |
| **+10%** | 友好环境 | 单位词条与地块相容 |
| **0** | 中立 | 普通地形，无加成减益 |
| **-10%** | 轻微不利 | 单位词条与地块不兼容 |
| **-20%** | 明显不利 | 单位词条与地块对立 |
| **-50%** | 极端克制 | 单位词条与地块严重冲突（如火词条站在水域） |

**伤害效果独立计算**：地形特殊效果（灼烧/溺水/冻结/困住等）的伤害单独计算，不在属性加成档位中。

> 详细交互矩阵见 [06_地形系统_完整](./06_地形系统_完整.md)

### 9.3 技能体系

| 类型 | 触发方式 | 示例 |
|------|----------|------|
| 主动技能 | 满足条件自动释放 | 突进、AOE、控制 |
| 被动技能 | 特定条件触发 | 相邻单位加成、地形触发 |
| 终极技能 | 能量槽满释放 | 大范围伤害 |

### 9.4 飞行单位

飞行单位是特殊的单位类型，具有以下特性：

| 特性 | 说明 |
|------|------|
| **地形加成** | ❌ 不受地形属性加成影响 |
| **地形负面效果** | ❌ 免疫所有地形负面效果（灼烧/溺水/冻结/困住等） |
| **移动限制** | ✅ 可通过任何地块，包括虚空 |
| **虚空移动** | ✅ 可在虚空上方移动 |
| **地块消失** | ✅ 不会坠毁，继续在虚空上方移动 |
| **连通性判定** | ❌ 不参与连通性判定（与地面单位相同） |

> **设计意图**：飞行单位作为高机动性单位，免疫地形效果是其核心价值，但不参与连通性判定保持游戏机制的一致性。

## 10. 战斗机制

### 10.1 交战规则

- 单位自动索敌、自动攻击
- 优先攻击攻击范围内的敌方单位
- 无敌人时向敌方基地移动
- 即时制，无回合概念

### 10.2 攻击目标优先级

1. 攻击范围内的敌方单位
2. 攻击范围内的敌方防御塔
3. 攻击范围内的敌方基地（需先摧毁至少一座防御塔）
4. 向敌方基地方向移动

### 10.3 伤害计算

```
最终伤害 = 基础攻击力 × 地形加成 × 技能加成 × 防御减免
```

### 10.4 死亡处理

- 单位生命归零时死亡
- 死亡单位从战场移除
- 部分单位有死亡触发效果

## 11. 大局进程（Roguelike爬塔）

### 11.1 节点系统

大局采用节点地图形式，玩家通过路线选择前进：

| 节点类型 | 图标 | 功能 |
|---------|:----:|------|
| 战斗 | ⚔️ | 敌方情报预览，确认后开始战斗 |
| 商店 | 🛒 | 购买/售卖地块和卡牌 |
| 休息 | 🏕️ | 回复基地血量 / 升级卡牌 / 移除卡牌（三选一） |
| 升级 | ⬆️ | 三选一：升级一张卡牌 |
| 精英 | 👑 | 强力敌人，击败后获得稀有遗物 |
| ? | ❓ | 随机事件、宝箱、或未知遭遇 |
| Boss | 👹 | Boss战，胜利后获得遗物 |

### 11.2 层次结构（3Act 45层）

| Act | 层数 | 敌人构成 | 精英怪物 | Boss |
|-----|------|----------|----------|------|
| Act 1 | 第1-15层 | 基础单位为主 | 3种 | 守门人（击败后进入Act 2） |
| Act 2 | 第16-30层 | 新增中型单位、组合战术 | 5种（含Act1精英+新增） | 梦境守护者（击败后进入Act 3） |
| Act 3 | 第31-45层 | 全套单位池、高难度AI | 8种（全部精英） | 最终Boss（需解锁才能挑战） |

**节点分布**（每层3-4个可选节点）：

| 阶段 | 战斗 | 商店 | 休息 | 升级 | 精英 | ? |
|------|:----:|:----:|:----:|:----:|:----:|:-:|
| 前期（2-4层） | 60% | 20% | 15% | - | - | 5% |
| 中期（5-9层） | 55% | 20% | 15% | 5% | - | 5% |
| 后期（10-14层） | 50% | 15% | 10% | 10% | 10% | 5% |
| Act2/3 | 50% | 15% | 10% | 5% | 15% | 5% |

### 11.3 奖励机制

| 层类型 | 胜利奖励 | 失败奖励 |
|--------|----------|----------|
| 普通层 | 金币 + 卡牌/地块三选一 | 少量金币 |
| 精英层 | 金币 + 稀有卡牌/地块/遗物三选一 | 少量金币 |
| Boss层 | 遗物（固定） + 大量金币 | 少量金币 |
| 失败 | **游戏结束**（经典Roguelike） | - |

### 11.4 新游戏初始流程

新游戏时进行多轮三选一配置：
- 进行5轮地块三选一
- 然后进行1-2轮卡牌三选一
- 玩家完全自由组合初始配置

## 12. 角色系统

### 12.1 核心功能

- **差异化初始配置**：每个角色有不同的4×7控制区地形分布
- **角色被动技能**：与单位/地形联动的被动效果
- **解锁后可选**：通过成就/进程解锁角色

### 12.2 基础角色（3个）

| 角色 | 风格定位 | 初始4×7控制区地形分布 | 被动技能 |
|------|----------|---------------------|---------------------|
| **元素师艾琳** | 元素连锁 | 水×8、火×4、森×4、草×12 | **元素共鸣**：相邻不同地形的单位各自获得+15%攻击力 |
| **守卫官雷恩** | 阵地防御 | 岩×8、草×12、森×4、农×4 | **坚守阵地**：己方单位在己方控制区内获得+25%防御力；消耗岩石地块后，相邻己方单位获得护盾 |
| **游侠希尔** | 机动突进 | 森×10、沙×6、草×12 | **森林猎手**：森林地形上的单位移速+30%；消耗森林地块后，该位置诞生临时侦察兵，持续10秒 |

### 12.3 解锁条件

| 角色 | 解锁条件 |
|------|----------|
| 元素师艾琳 | 初始可用 |
| 守卫官雷恩 | 到达第15层（击败Act1 Boss）解锁 |
| 游侠希尔 | 成就「一波流」完成解锁 |

### 12.4 角色专属卡牌

| 卡牌名称 | 限定角色 | 效果 |
|----------|----------|------|
| 元素爆发 | 艾琳 | 消耗3个相邻同地形地块，造成全场AOE伤害 |
| 要塞模式 | 雷恩 | 己方基地获得50%最大生命值的护盾，持续10秒 |
| 影袭 | 希尔 | 所有己方单位获得隐身3秒，首次攻击伤害翻倍 |

> 更多角色扩展设计见 [09_元游戏系统](./09_元游戏系统.md)

## 13. 局外成长系统

### 13.1 成就解锁制

保持Roguelike"每局独立"的纯粹性，同时通过成就解锁给予长期目标。

### 13.2 可解锁内容类型

| 内容类型 | 解锁示例 | 永久效果 |
|----------|----------|----------|
| 初始地块包 | "水系大师"成就解锁 | 新游戏初始三选一时可能出现高级地块 |
| 初始卡牌包 | "法术大师"成就解锁 | 新游戏初始卡牌三选一可能出现稀有卡 |
| 初始配置方案 | "速通专家"成就解锁 | 新游戏时可选预设开局方案 |
| 新卡牌 | 达成特定成就 | 40张基础卡池之外的额外卡牌 |
| 新角色 | 达成特定成就/进程 | 解锁新角色供选择 |
| 真结局路线 | 击败普通Boss | 下次游戏出现特殊事件分支 |

> 详细成就列表见 [09_元游戏系统](./09_元游戏系统.md)
