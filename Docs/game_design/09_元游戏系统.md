## 一、用户确认的设计方向

| 模块 | 选择方向 |
|------|----------|
| 局外成长 | **成就解锁制** - 达成特定成就可永久解锁新内容（卡牌/地块/初始配置） |
| Roguelike模式 | **杀戮尖塔结构** - 3大Act，每Act15层，共45层 |
| 无尽塔 | **删除** - 不需要无尽模式 |
| 天梯系统 | **删除** - 纯单机游戏 |
| 每日挑战 | 固定种子 + 固定初始配置 + 特殊规则三合一 |
| 成就系统 | **混合分类** - 进程成就 + 行为成就 + 隐藏成就 |
| 角色系统 | 差异化初始配置 + 解锁后可选 + 角色被动技能 |
| 角色数量 | **3个基础角色**（后续扩展） |

## 二、局外成长系统

### 核心定位

保持Roguelike"每局独立"的纯粹性，同时通过成就解锁给予长期目标。每次新游戏仍从零开始，但可选择的"起跑线"会随着成就解锁而越来越丰富。

### 可解锁内容类型

| 内容类型 | 解锁示例 | 永久效果 |
|----------|----------|----------|
| 初始地块包 | "水系大师"成就解锁 | 新游戏初始三选一时可能出现高级地块 |
| 初始卡牌包 | "法术大师"成就解锁 | 新游戏初始卡牌三选一可能出现稀有卡 |
| 初始配置方案 | "速通专家"成就解锁 | 新游戏时可选预设开局方案 |
| 新卡牌 | 达成特定成就 | 40张基础卡池之外的额外卡牌 |
| 新角色 | 达成特定成就/进程 | 解锁新角色供选择 |
| 真结局路线 | 击败普通Boss | 下次游戏出现特殊事件分支 |

## 三、Roguelike模式结构

### 整体规模

```
┌─────────────────────────────────────────────────────┐
│  Act 1（第1-15层）                                  │
│  • 敌人构成：基础单位为主                            │
│  • 精英怪物：3种                                    │
│  • Boss：守门人（击败后进入Act 2）                   │
├─────────────────────────────────────────────────────┤
│  Act 2（第16-30层）                                 │
│  • 敌人构成：新增中型单位、组合战术                   │
│  • 精英怪物：5种（含Act1精英+新增）                  │
│  • Boss：梦境守护者（击败后进入Act 3）               │
├─────────────────────────────────────────────────────┤
│  Act 3（第31-45层）                                 │
│  • 敌人构成：全套单位池、高难度AI                     │
│  • 精英怪物：8种（全部精英）                         │
│  • 最终Boss：需解锁才能挑战                          │
└─────────────────────────────────────────────────────┘
```

### 节点分布（每层3-4个可选节点）

| 阶段 | 战斗 | 商店 | 休息 | 升级 | 精英 | ? |
|------|:----:|:----:|:----:|:----:|:----:|:-:|
| 前期（2-4层） | 60% | 20% | 15% | - | - | 5% |
| 中期（5-9层） | 55% | 20% | 15% | 5% | - | 5% |
| 后期（10-14层） | 50% | 15% | 10% | 10% | 10% | 5% |
| Act2/3 | 50% | 15% | 10% | 5% | 15% | 5% |

### 节点类型说明

| 节点类型 | 图标 | 说明 |
|----------|:----:|------|
| 战斗 | ⚔️ | 与敌人对战，获胜后获得奖励（金币/卡牌/地块） |
| 商店 | 💰 | 使用金币购买卡牌、地块、遗物 |
| 休息 | 🏕️ | 恢复基地血量、升级卡牌、或移除卡牌 |
| 升级 | ⬆️ | 三选一：升级一张卡牌 |
| 精英 | 👑 | 强力敌人，击败后获得稀有遗物 |
| ? | ❓ | 随机事件、宝箱、或未知遭遇 |

## 四、每日挑战模式

### 核心概念

固定种子 + 固定初始配置 + 特殊规则，所有玩家在同一起跑线上竞争。

### 挑战构成

| 组成部分 | 说明 |
|----------|------|
| **固定种子** | 地图节点、敌人配置、商店刷新、事件结果完全一致 |
| **固定初始配置** | 指定5轮地块三选一的结果，所有人起手相同 |
| **特殊规则** | 每日一个规则修饰符，改变游戏体验 |

### 特殊规则列表

| 规则名称 | 效果 |
|----------|------|
| 水系主宰 | 所有水地块召唤单位生命+50%，其他地形-20% |
| 法力暴涨 | 初始法力8点，回复速度每秒+2 |
| 地块匮乏 | 初始地块库存减半，但每块地块可召唤2次 |
| 精英盛宴 | 每3层必定是精英层 |
| 双倍金币 | 所有金币收益×2，但商店价格×1.5 |
| 卡牌雨 | 初始卡牌选择从2轮增加到4轮 |
| 无地形 | 初始控制区全部为空白地块，需战斗中争夺 |

### 排行榜

记录以下数据，三项综合评分：

| 评分项 | 权重 |
|--------|------|
| 最高层数 | 50% |
| 用时 | 30% |
| 最终基地血量 | 20% |

## 五、成就系统

### 分类：进程成就 + 行为成就 + 隐藏成就

#### 5.1 进程成就（推进类）

| 成就名称 | 解锁条件 | 解锁奖励 |
|----------|----------|----------|
| 初入战场 | 完成第1层 | 解锁：初始地块包A（含水、沙、草） |
| 站稳脚跟 | 到达第5层 | 解锁：初始卡牌包A（含3张普通卡） |
| 中盘站稳 | 到达第15层（Act1 Boss） | 解锁：新初始配置「速攻流」 |
| 梦境穿越 | 到达第30层（Act2 Boss） | 解锁：新初始配置「防御流」 |
| 击败守门人 | 首次击败Act1 Boss | 解锁：真结局路线触发条件 + 角色「守卫官雷恩」 |
| 击败梦守护 | 首次击败Act2 Boss | 解锁：最终Boss挑战权 |

#### 5.2 行为成就（技巧类）

| 成就名称 | 解锁条件 | 解锁奖励 |
|----------|----------|----------|
| 断路大师 | 单局造成3次敌方断路 | 解锁：卡牌「断路陷阱」 |
| 一波流 | 单局在60秒内获胜 | 解锁：角色「游侠希尔」 |
| 完美无伤 | 单局基地血量>80%获胜 | 解锁：地块「神圣祭坛」 |
| 地块大师 | 单局消耗50个地块 | 解锁：卡牌「地块再造」 |
| 法术狂人 | 单局使用15张卡牌 | 解锁：卡牌「无限法力」 |

#### 5.3 隐藏成就（探索类）

| 成就名称 | 解锁条件 | 解锁奖励 |
|----------|----------|----------|
| ??? | 在特定事件节点做出特定选择 | 解锁：特殊地块「虚空水晶」 |
| ??? | 使用特定卡牌组合 | 解锁：传说卡牌「空间置换」 |

## 六、角色系统

### 核心功能

- **差异化初始配置**：每个角色有不同的4×7控制区地形分布
- **角色被动技能**：与单位/地形联动的被动效果
- **解锁后可选**：通过成就/进程解锁角色

### 基础角色（3个）

| 角色 | 风格定位 | 初始4×7控制区地形分布 | 被动技能 |
|------|----------|---------------------|---------------------|
| **元素师艾琳** | 元素连锁 | 水×8、火×4、森×4、草×12 | **元素共鸣**：相邻不同地形的单位各自获得+15%攻击力 |
| **守卫官雷恩** | 阵地防御 | 岩×8、草×12、森×4、农×4 | **坚守阵地**：己方单位在己方控制区内获得+25%防御力；消耗岩石地块后，相邻己方单位获得护盾 |
| **游侠希尔** | 机动突进 | 森×10、沙×6、草×12 | **森林猎手**：森林地形上的单位移速+30%；消耗森林地块后，该位置诞生临时侦察兵，持续10秒 |

### 解锁条件

| 角色 | 解锁条件 |
|------|----------|
| 元素师艾琳 | 初始可用 |
| 守卫官雷恩 | 到达第15层（击败Act1 Boss）解锁 |
| 游侠希尔 | 成就「一波流」完成解锁 |

### 角色专属卡牌

| 卡牌名称 | 限定角色 | 效果 |
|----------|----------|------|
| 元素爆发 | 艾琳 | 消耗3个相邻同地形地块，造成全场AOE伤害 |
| 要塞模式 | 雷恩 | 己方基地获得50%最大生命值的护盾，持续10秒 |
| 影袭 | 希尔 | 所有己方单位获得隐身3秒，首次攻击伤害翻倍 |
