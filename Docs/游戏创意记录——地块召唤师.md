# 游戏创意记录——地块召唤师

# 游戏策划案：地块召唤师 (Tile Summoner)

## 1. 核心玩法概述

本游戏为结合**策略自走棋**与**卡牌构建**的对战游戏。核心特色在于将“地块”作为基础资源，通过地形编辑、单位召唤与连通性规则，进行动态的战场博弈。

## 2. 地图与地形系统

### 地图结构

- **规格**：7×9 非对称地图。
- **布局**：通过非对称设计制造进攻路线与防守重点的差异，敌方采用水平镜像布局。

### 地形交互

- **类型**：水、沙、岩石、草地、森林、农田、熔岩、沼泽、冰原。
- **生成机制**：每种地形可生成 3 种对应单位（前排、防守辅助、功能性单位）。
- **地形编辑**：玩家半区地块视为“基础资源”，可被消耗用于召唤，或进行编辑改造。

## 3. 战斗规则与机制

### 胜负条件

- **目标**：摧毁敌方主基地。
- **前置条件**：需先摧毁至少一个防御塔（位于 (2, 8) 或 (6, 8)）才能攻击主基地（位于 (4, 9)）。

### 核心机制：连通性与虚空

- **主路规则**：必须存在一条从敌方基地连通至另一方主基地的路径。
- **断路判负**：操作导致断路的一方立即判负。
- **坠入虚空**：
    1. 以“主路”为基准进行连通性检查。
    2. 无法连通回主路的地形格及其上的非飞行单位将“坠入虚空”消失。
    3. **战术意图**：鼓励玩家平衡“掘地/改路”的高风险与高收益。

### 战场控制区域

- **控制带**：地图按“行”划分。
- **夺取规则**：在敌方半区某行，我方单位数量持续 5 秒多于敌方（含相邻行容差）。
- **前线优势**：
    - 获取控制权后，可消耗该行地块召唤单位。
    - **代价梯度**：距离己方越远，召唤成本（法力/地块消耗）越高。
    - **战略意义**：推动战线，实现前置压制。

## 4. 经济与资源循环

| 资源类型 | 获取/恢复 | 用途 |
| --- | --- | --- |
| :--- | :--- | :--- |
| **法力值** | 每 3 秒 1 点 | 召唤单位的基础消耗 |
| **玩家地块** | 初始持有/争夺 | 召唤单位的“物质”消耗；地形改造 |
| **金币** | 战斗产出 | 卡组升级、养成 |
| **卡牌** | 牌库抽取 | 消耗品；提供数值强化、范围效果或特效 |
| **遗物** | 局外/特殊掉落 | 提供被动/主动技能；定义战术流派（防守/爆发/运营） |

## 5. 单位系统

- **属性加成**：依据所在地形获得移动力、攻防血量或特效概率加成。
- **技能体系**：
    - **主动**：突进、位移、AOE、控制。
    - **被动**：特定地形或相邻单位组合触发。
- **AI 驱动**：自动索敌与移动；优先级受卡牌/遗物/Buff 修正。
- **飞行单位**：无视地形增益/减益；不受“地形消失”直接影响（但可被技能移除）。

## 6. 待补充必要内容 (Missing Content)

### 基础架构缺失

- **UI/交互逻辑**：地块选择方式、卡牌释放操作、移动端/PC端适配方案。
- **具体数值表**：各单位的基础属性、地形加成具体数值、法力/金币消耗曲线。
- **卡池设计**：具体的卡牌清单（名称、效果、费用）、卡组构建规则（上限、颜色/阵营限制）。

### 玩法闭环缺失

- **单局流程**：回合制还是纯即时制？（目前仅提及“每3秒回蓝”，倾向即时或半即时）。
- **局外成长**：卡组与遗物的获取路径、Roguelike 元素具体实现、Meta 进度。
- **世界观/包装**：召唤师背景、地形生成的逻辑解释、阵营对立设定。

### 表现层缺失

- **视觉反馈**：“坠入虚空”的表现效果、地形变化的动画、控制权切换的UI提示。
- **音效需求**：环境音、单位反馈、核心事件（断路预警）音效。
